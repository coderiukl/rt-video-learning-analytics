# Design and Implementation of a Real-Time Video Learning Behavior Analytics System

## Project Overview
This project aims to design and implement a real-time system that analyzes student behavior while watching educational videos.

The system collects user interaction events such as:

- Play
- Pause
- Seek
- Replay
- Watch duration

These events are processed to generate learning analytics insights including:

- Student engagement analysis
- Video heatmaps
- Learning behavior patterns
- Dropout prediction

## Technologies
- Django
- Python
- PostgreSQL
- JavaScript
- WebSocket
- Machine Learning (Scikit-learn)

## System Architecture

Video Player → Event Tracking → Backend API → Real-time Analytics → Dashboard
